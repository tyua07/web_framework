参数 默认值 描述
title null (必须) 弹出框标题（可以是html格式）
text null 弹出框文本内容（可以是html格式）
html false 如果设置为true，将无法逃避的标题和文本参数。（如果你担心XSS攻击。设置为false）
type null 显示相应的图标动画：“警告”，“错误”，“成功”和“信息”（类型：warning(警告)；error（错误）;info(信息);success(成功)）
allowEscapeKey true 如果设置为true，用户可以按Escape键关闭弹出框。
customClass null 自定义的CSS类弹出框。
allowOutsideClick false 如果设置为true，用户可以通过点击外面关闭弹出框。
showCancelButton false 如果设置为TRUE，“取消”按钮将显示，用户可点击关闭弹出框。
showConfirmButton true 如果设置为FALSE，“确定/确认”按钮将被隐藏
confirmButtonText "OK" 改变确认按钮文本内容
confirmButtonColor "#AEDEF4" 改变确认按钮背景颜色（必须是一个十六进制值）。
cancelButtonText "Cancel" 改变取消按钮文本内容
closeOnConfirm true 是否关闭确认按钮
imageUrl null 添加一个自定义图标的弹出框
imageSize "80x80" 如果源设置，您可以指定如何大你想要的图标是在PX图像大小。传递一个字符串值分离的一个“X”。第一个值是宽度，二是高度。
timer null 弹出框自动关闭定时器。在MS（毫秒）。
animation true 如果设置为FALSE，弹出框的动画将被禁用。
